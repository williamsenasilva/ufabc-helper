UFABC Helper

Sobre

Aplicação web onde o usuário entra com sua ficha de aluno e o sistema retorna o calculo dos coeficientes CR, CA e CP - como o Portal do Aluno faz. O diferencial da aplicação é que nela o usuário pode inserir as disciplinas que está cursando e simular o cálculo dos coeficientes dependendo da nota dessas disciplinas.

Funcionamento

O aluno entra no Portal do Aluno e pega o json que é fornecido pelo próprio portal. Como na imagem abaixo.

O json fornecido traz todas as disciplinas já cursadas com notas, quantidade de créditos, etc. Como no exemplo abaixo. 

          [
            {"periodo":"2","ano":2014,"codigo":"BCS0001-13","situacao":"Aprovado","categoria":"Obrigatória","creditos":3,"disciplina":"Base Experimental das Ciências Naturais","conceito":"A"},
            {"periodo":"2","ano":2014,"codigo":"BIL0304-13","situacao":"Aprovado","categoria":"Obrigatória","creditos":3,"disciplina":"Origem da Vida e Diversidade dos Seres Vivos","conceito":"C"}
          ]

Após o aluno baixar esse json ele deve entrar na aplicação web e inserir este json. A aplicação calcula CR, CA e CP e permite inclusão de novas disciplinas ou edição das existentes, resultando em novos valores dos coeficientes.

Desenvolvimento

  • Python - interface web
  • MariaDB - banco de dados da aplicação
  • Git - versionamento
